Task Management Application

A full-stack Task Management Application built using the MERN Stack (MongoDB, Express.js, React, Node.js) that allows users to manage their daily tasks efficiently. Users can create, update, delete, and track the status of tasks with a simple and responsive interface.

Features:

1.User authentication and login

2.Create new tasks

3.View all tasks created by the user

4.Update task details

5.Delete tasks

6.Mark tasks as completed

7.Completed tasks are visually highlighted

8.Task deadline management

9.Responsive user interface

Tech Stack:
->Frontend

1.React

2.Redux

3.Bootstrap

4.Axios

->Backend

1.Node.js

2.Express.js

->Database

1.MongoDB

2.Mongoose

To run the front end server:
npm run dev

To run the Backend express Server:
node server.js
